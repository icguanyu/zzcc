VC101
自然生態輕鬆行

  • 課程編號：

    VC101

  • 講師姓名：

    林金松

  • 講師學歷：

    新埔工專

  • 講師學歷：

    電腦公司維護工程師、程式設計師、主管 SUN電腦訓練中心 Java 程式設計系列課程講師 荒野保護協會解說員訓練講師及輔導員 荒野保護協會推廣講師訓練講師 社區大學講師

  • 上課時段：

    每週一 上午 09:30-12:00

  • 課程理念：

    藉由自然體驗、關懷土地的理念出發，探索人與自然的關係，讓學員能真正地愛好自然， 願意分享在自然中感受到的美好及喜悅， 進而對自然環境的破壞產生疼惜與感觸， 而積極參與守護環境。

  • 教學方式：

    室內課以照片及演說方式，搭配戶外課的實際觀察體驗。室內課在週一上午，戶外課在週一上午或全天。

  • 評鑑方式：

    出席率40%，良善對待環境態度60%

  • 參考書目：

    徐仁修著作系列作品、大樹自然系列書籍

  • 選課要求：

    能不需旁人協助下參與戶外活動，不需要走很快，也不建議走很快。

  • 學分費用：

    3000元

  • 額外費用：

    戶外課程的各項費用，如: 食宿費、交通費、門票及其它分攤之費用等。交通以共乘或租車為主，若共乘則由乘客補貼開車學員費用。

  • 招生人數：

    22人以上始開班

  • 課程大綱：

    ※僅供學員參考，實際授課內容視學員需求調整
